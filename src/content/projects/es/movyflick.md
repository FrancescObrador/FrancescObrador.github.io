Construida con <strong>React</strong> y <strong>TypeScript</strong> para una arquitectura de componentes con tipado estricto, <strong>Redux</strong> para la gestión del estado global y <strong>Firebase</strong> para persistencia de datos en tiempo real y autenticación. Aplica patrones orientados a producción — estado tipado, flujos de datos asíncronos y composición de componentes — en un proyecto personal completo.
